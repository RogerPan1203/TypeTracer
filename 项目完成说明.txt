##如何使用

### 1. 打开项目
```bash
open /Users/hyc/Documents/Coding/TypeTracer/TypeTracer.xcodeproj
```

### 2. 在Xcode中运行
- 按 `Cmd + R` 或点击运行按钮
- 选择 "My Mac" 作为目标设备

### 3. 授予权限
应用首次运行时：
1. 点击"请求辅助功能权限"按钮
2. 在系统设置中勾选 TypeTracer
3. 返回应用，自动开始监控

### 4. 测试功能
- 打开任意文本编辑器
- 输入一些文字
- 观察TypeTracer中的统计数据更新

##  界面预览说明

### 顶部状态栏
- 左侧：应用标题 + 监控状态（绿色/红色圆点）
- 右侧：总按键数统计

### 中间内容区
- **实时统计标签页**：
  - 三个统计卡片（蓝色、绿色、橙色）
  - 每小时柱状图（0-23小时）
  
- **历史数据标签页**：
  - 最近7天的数据列表
  - 每行显示日期、条形图、数字

### 底部控制栏
- 请求权限按钮（橙色，仅在无权限时显示）
- 清除数据按钮（灰色）
- 开始/停止监控按钮（蓝色）


##重要提示

### 必须配置的权限
应用需要辅助功能权限才能工作，配置路径：
```
 → 系统设置 → 隐私与安全性 → 辅助功能 → ✓ TypeTracer
```

### App Sandbox
项目的 `TypeTracer.entitlements` 文件已配置为禁用 App Sandbox，这是必需的，因为：
- 全局键盘监听需要系统级权限
- App Sandbox会阻止访问辅助功能API

### 数据存储
- 数据存储在 UserDefaults 中
- 自动保留最近7天的数据
- 可以通过"清除数据"按钮重置


## ✨ 特别说明

1. **权限是必需的**：应用必须获得辅助功能权限才能工作
2. **不记录内容**：应用只记录按键时间，不记录任何输入内容
3. **本地存储**：所有数据存储在本地，不上传到任何服务器
4. **实时更新**：统计数据每秒自动更新

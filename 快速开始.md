# TypeTracer 快速开始指南

## 第一次使用

### 步骤 1: 打开项目
```bash
cd /Users/hyc/Documents/Coding/TypeTracer
open TypeTracer.xcodeproj
```

### 步骤 2: 运行应用
1. 在 Xcode 中按 `Cmd + R` 运行应用
2. 应用会打开一个窗口

### 步骤 3: 授予权限
1. 点击应用底部的"请求辅助功能权限"按钮（橙色）
2. 系统会自动打开"系统设置"
3. 在"隐私与安全性 → 辅助功能"中
4. 找到并勾选 **TypeTracer**
5. 返回应用，会自动开始监控

### 步骤 4: 开始使用
1. 打开任意文本编辑器（如备忘录）
2. 随便输入一些文字
3. 返回 TypeTracer 查看统计数据
4. 观察"最近一分钟"的数字变化

## 界面说明

### 实时统计页面
- 🔵 最近一分钟 - 过去60秒的按键数
- 🟢 最近一小时 - 过去60分钟的按键数
- 🟠 今日 - 今天0点到现在的按键数
- 📊 每小时柱状图 - 显示今天每小时的打字量

### 历史数据页面
- 显示最近7天的打字统计
- 包含日期、条形图和具体数字

## 常见问题

**Q: 为什么没有开始统计？**
A: 确保已经授予辅助功能权限，并且状态指示器显示绿色（正在监控）

**Q: 数据会保存吗？**
A: 会！数据自动保存，重启应用后会自动加载

**Q: 如何重置数据？**
A: 点击底部的"清除数据"按钮

**Q: 应用会记录我输入的内容吗？**
A: 不会！只记录按键次数和时间，不记录任何内容
